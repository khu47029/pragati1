# Pragati Sahayak 1000 Website Add-ons Pack

This pack includes the previous **299 Ultra Website Add-ons files** and adds a larger premium layer for real website use.

## How to install

1. Extract this ZIP.
2. Open the folder `pragati-sahayak-1000-website-addons`.
3. Copy the inner folders into your existing GitHub repository root.
4. Do not delete your existing website files.
5. Commit and deploy on GitHub Pages.

## Add to `index.html` head

```html
<link rel="stylesheet" href="assets/css/ps-ultra-system.css">
<link rel="stylesheet" href="assets/css/ps-components.css">
<link rel="stylesheet" href="assets/css/ps-dashboard.css">
<link rel="stylesheet" href="assets/css/ps-1000-system.css">
<script defer src="assets/js/ps-ultra-core.js"></script>
<script defer src="assets/js/ps-1000-core.js"></script>
```

## Homepage section

Copy the code from:

`integration-snippets/homepage-1000-growth-system.html`

Paste it inside `index.html` where you want the new premium section.

## Important

This add-on pack does not change your logo, phone number, WhatsApp number, Instagram, Twitter/X, or UPI ID.
