# How to Publish This Updated Website

1. Extract this ZIP.
2. Upload/replace all files in your GitHub repository.
3. Keep GitHub Pages enabled for the main branch.
4. After publishing, open:
   - https://pragatisahayak.in/sitemap.xml
   - https://pragatisahayak.in/robots.txt
   - https://pragatisahayak.in/llms.txt
5. Submit the sitemap in Google Search Console.
6. Test important pages on mobile.
7. For future content, add one useful blog every week or month.

Important: Existing phone, WhatsApp, UPI, Instagram and X/Twitter details were preserved.
