# Copy Order

1. Copy `assets/` first.
2. Copy `pages/` second.
3. Copy `components/`, `data/`, `schemas/`, `content/`, and `templates/`.
4. Copy `integration-snippets/` and paste required snippets into `index.html`.
5. Optional: copy `.github/` workflows if you want repo checks.
6. Commit in GitHub.

Important: If GitHub asks to replace files, check if it is your old file. This pack uses new names to avoid accidental replacement.
